# Agentic-AI

## What are AI Agents?

AI Agents are AI systems that don't just respond to a single prompt, they perceive their environment, reason about a goal, and take a sequence of actions (often using tools) to achieve it, adapting as new information comes in.

In short: an AI Agent = a language model + memory/context + tools + a feedback loop that lets it plan and act autonomously toward a goal, rather than just generating a one-off reply.

Example: an assistant like me (Claude) becomes "agentic" when it can search the web, call APIs, browse a page, or run code on its own to complete a multi-step task instead of only answering from what it already knows.
